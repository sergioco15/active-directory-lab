# active-directory-lab
Active Directory Home Lab &amp; Security Monitoring A comprehensive on-premises deployment featuring a fully functional Windows Domain. This project covers the end-to-end installation of Active Directory, automated logging via Splunk, and the configuration of Windows workstations and servers for security auditing.
## 🛠️ Technologies Used- Windows Server- Active Directory- VirtualBox - Cisco Packet Tracer
